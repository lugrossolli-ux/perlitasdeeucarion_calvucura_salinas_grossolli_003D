# perlitas_de_eucarion
"Perlitas de Eucarion", pyme de bisutería, solucionó sus problemas de gestión manual con una app propia. El sistema centraliza inventario, ventas, encargos y reportes. Así, simplifica tareas, reduce errores y brinda a la dueña una visión organizada para controlar y proyectar mejor su negocio.
